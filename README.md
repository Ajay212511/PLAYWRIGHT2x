# PLAYWRIGHT2x

A personal learning repository for JavaScript basics, concepts, and development shortcuts.

## About

This repository contains step-by-step JavaScript learning materials, code examples, and quick reference guides. It is designed to build a strong foundation in JavaScript before moving on to advanced topics like Playwright automation testing.

## Repository Structure

```
PLAYWRIGHT2x/
├── Basic_01/
│   ├── 01_basic.js              # Node.js process basics (platform, arch, version)
│   ├── 01_basicWord.js          # Basic JavaScript words/concepts
│   ├── 01_Core_Comments.js      # Single-line and multi-line comments
│   ├── 01_hotcode.js            # Hot code / quick snippets
│   ├── 01_identifier.js         # Introduction to identifiers, literals, operators
│   ├── 01_06_Var_let.js         # var vs let differences
│   ├── 01_07_Identifier.js      # Complete identifier rules guide with examples
│   └── 03_Shortcut.md           # Windows shortcuts for JavaScript development
└── README.md                    # This file
```

## Topics Covered

### JavaScript Fundamentals
- **Identifiers & Literals** -- Naming rules, valid/invalid examples, best practices
- **Comments** -- Single-line (`//`) and multi-line (`/* */`) comments
- **Variables** -- `var` vs `let`, declaration rules, and usage
- **Operators** -- Assignment and arithmetic operators
- **Case Sensitivity** -- Understanding JavaScript's case-sensitive nature

### Development Tools
- **Windows Shortcuts** -- Browser DevTools, VS Code, Terminal, and General Windows shortcuts
- **Node.js Basics** -- Running JavaScript files via command line

## How to Run

1. Make sure you have [Node.js](https://nodejs.org/) installed.
2. Open a terminal in the project folder.
3. Run any JavaScript file:

```bash
node Basic_01/01_07_Identifier.js
```

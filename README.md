## About

This repository is a step-by-step JavaScript learning journey, building from basic syntax all the way to browser automation with Playwright. Each chapter focuses on a specific concept with practical, real-world code examples and interview-style IQ questions.

## How to Run

1. Make sure you have [Node.js](https://nodejs.org/) installed.
2. Open a terminal in the project folder.
3. Run any JavaScript file:

```bash
node Basic_01/01_07_Identifier.js
```

For Playwright chapters (Chapter_17), first install dependencies:

```bash
cd Chapter_17_Playwright
npm install
```

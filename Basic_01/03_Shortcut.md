# Windows Shortcuts for JavaScript Development

A quick reference guide for the most useful keyboard shortcuts when working with JavaScript on Windows.

---

## 🌐 Browser DevTools (Console)

| Shortcut | Action |
|----------|--------|
| `F12` | Open/Close DevTools |
| `Ctrl + Shift + J` | Open Console directly |
| `Ctrl + Shift + I` | Open DevTools (Inspector) |
| `Ctrl + Shift + C` | Inspect Element tool |
| `Ctrl + L` | Clear console |
| `Ctrl + Enter` | Run code in console (multiline) |

---

## 💻 VS Code (Most Common JS Editor)

| Shortcut | Action |
|----------|--------|
| `` Ctrl + ` `` | Open Integrated Terminal |
| `Ctrl + Shift + P` | Command Palette |
| `Ctrl + /` | Toggle line comment (`//`) |
| `Shift + Alt + A` | Toggle block comment (`/* */`) |
| `F5` | Start Debugging (run JS) |
| `Ctrl + F5` | Run without Debugging |
| `F2` | Rename symbol (variable/function) |
| `Ctrl + Space` | Trigger IntelliSense (auto-complete) |
| `Ctrl + D` | Select next occurrence |
| `Alt + Shift + F` | Format document |
| `Ctrl + Shift + L` | Select all occurrences |

---

## 🖥️ Windows Terminal / CMD / PowerShell

| Shortcut | Action |
|----------|--------|
| `Ctrl + C` | Stop running Node.js script |
| `Ctrl + L` or `cls` | Clear screen |
| `↑` (Up Arrow) | Previous command |
| `Tab` | Auto-complete file/folder names |
| `node filename.js` | Run a JavaScript file |

---

## ⌨️ General Windows

| Shortcut | Action |
|----------|--------|
| `Win + .` or `Win + ;` | Emoji picker |
| `Win + V` | Clipboard history |
| `Win + Shift + S` | Snipping Tool (screenshots) |
| `Alt + Tab` | Switch between windows |

---

## 🚀 Quick Run JS File (Node.js)

```bash
# In terminal, navigate to your folder
cd C:\Playwright2x\Basic_01

# Run any file
node 01_07_Identifier.js
```

**Tip:** In VS Code, right-click in the editor and select **"Run Code"** (if you have the Code Runner extension installed) -- shortcut is `Ctrl + Alt + N`.

---

## 📌 Pro Tips

1. **Learn `Ctrl + Shift + P`** -- it gives you access to every command in VS Code.
2. **Use `Ctrl + D`** to quickly refactor variable names across multiple lines.
3. **Master `Ctrl + /`** for quickly enabling/disabling lines of code while debugging.
4. **`F12`** is your best friend for instantly inspecting any webpage.
